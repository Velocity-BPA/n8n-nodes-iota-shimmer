# Automate IOTA and Shimmer with n8n: Introducing Our New Community Node

We're excited to announce the release of our latest n8n community node: n8n-nodes-iota-shimmer! Created by Velocity BPA, this node brings the power of IOTA and Shimmer networks directly into your n8n workflows.

## Why IOTA/Shimmer Automation Matters

IOTA and Shimmer represent the next generation of distributed ledger technology, offering feeless transactions and innovative features like native tokens, decentralized identifiers (DIDs), and data streams. However, integrating these capabilities into business workflows has traditionally required deep technical expertise and custom development.

That's where our new node comes in. Whether you're building IoT solutions, managing digital identities, or automating token operations, you can now leverage IOTA and Shimmer's capabilities through n8n's intuitive visual interface—no blockchain expertise required.

## Key Features

Our n8n-nodes-iota-shimmer node provides comprehensive access to IOTA and Shimmer networks:

- **Feeless Transactions**: Send and receive value without transaction fees, perfect for micropayments and IoT use cases
- **IRC-30/27 Token Support**: Create, mint, and manage native tokens on the network
- **Decentralized Identifiers (DIDs)**: Build self-sovereign identity solutions and credential management systems
- **Streams Integration**: Access IOTA Streams for secure, authenticated data channels

These features open up countless automation possibilities—from triggering workflows based on token transfers to automating identity verification processes or publishing IoT sensor data to immutable streams.

## Getting Started

Installing the node is simple. In your n8n instance, run:


npm install n8n-nodes-iota-shimmer


After restarting n8n, you'll find the IOTA/Shimmer node in your node palette, ready to integrate into your workflows.
